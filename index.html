NoiseScape es una herramienta para simular el sonido del viento, pero con las propiedades de los intervalos y escalas musicales. Utilizando el teclado, se pueden activar las notas como si fueran teclas de piano, para obtener un viento tenebroso, triste, alegre, etc.

Los controles permiten:
- Modular la amplitud (Amp.), frecuencia (Freq.) y ancho de banda (Q) del oscilador.
- Subir o bajar de octava (teclas Z y X).
- Mantener las notas activas (C).
- Grabar y descargar un archivo de audio.

* App solo disponible para escritorio (por ahora).
